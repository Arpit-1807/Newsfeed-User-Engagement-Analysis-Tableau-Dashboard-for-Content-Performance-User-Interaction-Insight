# Newsfeed-User-Engagement-Analysis | Tableau-Dashboard-for-Content-Performance-User-Interaction-Insight

1. **Project Overview**

This project analyzes user interaction patterns with a digital learning platform’s Newsfeed feature to understand how different post types drive engagement.

Using **Tableau**, an interactive dashboard was built to analyze **posts, likes, comments, visits, and user activity patterns** between **January 2023 and May 2023**.

The goal of this analysis is to identify:

• Which **content types generate the highest engagement**

• How **user interactions evolve over time**

• What **content strategies maximize platform engagement**

The findings help improve content prioritization, user retention, and engagement strategy.

2. **Business Objective**

The key objectives of this analysis were to:

• Evaluate **user engagement metrics across the Newsfeed**

• Identify **high-performing content types**

• Analyze **posting trends and interaction patterns**

• Measure **engagement through likes, comments, and visits**

• Provide **data-driven recommendations for improving Newsfeed performance**

4. **Dashboard Insights**

A. **Monthly Post Activity & Average Likes**

| Month    | Posts      | Avg Likes         |
| -------- | ---------- | ----------------- |
| January  | 8,665      | 5.16              |
| February | 7,235      | 4.54              |
| March    | **17,933** | 4.18              |
| April    | 5,838      | **3.82 (lowest)** |
| May      | 8,540      | 4.64              |

**Insight**

• **March recorded the highest posting activity (17,933 posts)**.

• **Average engagement dropped as post volume increased**, indicating possible **content saturation**.

**Business Interpretation**

Increasing post frequency does not necessarily improve engagement. Optimizing **content quality and timing** may deliver better interaction rates.

B. **Overall User Engagement**

| Metric       | Count     |
| ------------ | --------- |
| Visits       | **3,956** |
| Active Users | **405**   |
| Likes        | **362**   |
| Comments     | **111**   |
| Posts        | **47**    |

**Insight**

• High **visit count vs engagement actions** suggests many users **view posts but interact less frequently**.

**Business Interpretation**

Improving **interactive content formats** may increase engagement rates.

**Insight**

Although rare, **Text Posts drive the most discussions.**

**Business Interpretation**

Encouraging **community discussions and user-generated content** may improve engagement.

5. **Key Business Insights**

• Course Certificate posts generate the highest likes (1,176)

• Text posts drive the most conversations (73 comments)

• Automated milestone posts dominate the feed

• Higher post volume does not guarantee higher engagement

• 3,956 visits resulted in relatively low interaction actions

6. **Strategic Recommendations**

A. **Increase User-Generated Content**

Encourage more **text posts and discussions** to drive community interaction.

B. **Prioritize Achievement-Based Content**

Highlight **course certificates and milestone achievements**, which generate the highest likes.

C. **Balance Automated vs Manual Content**

Reduce excessive automated posts to maintain **content diversity and engagement quality**.

D. **Optimize Posting Frequency**

Avoid content saturation by scheduling posts strategically
